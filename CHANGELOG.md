# Changelog

## [2.0.0] - 2026-02-20

### Added
- Novo sistema de decisão hierárquico (4 níveis)
- Triagem crítica obrigatória como primeiro passo
- Componentes otimizados para uso sob pressão
- Sistema de design tokens
- Persistência local de progresso
- Feature flag para alternar entre V1 e V2

### Changed
- Arquitetura de tipos TypeScript completamente reestruturada
- Interface visual com hierarquia clara (pergunta > botões > ajuda)

### Deprecated
- Sistema antigo de decisão (FlowNode, FlowOption) será removido em v3.0.0
